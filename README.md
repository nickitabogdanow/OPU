# OPU TL.0018 azimuth control

Скрипт для управления опорно-поворотным устройством ТехЛазер TL.0018 по Ethernet.

Сейчас реализован проход по азимуту: скрипт последовательно отправляет целевые углы, ждет достижения каждой позиции и делает паузу между точками.

## Требования

- Python 3.10 или новее.
- ОПУ подключено по Ethernet и отвечает на `ping`.
- В веб-интерфейсе ОПУ работает управление.
- Открыт сервисный TCP-порт управления. По умолчанию используется `9760`.

Внешние Python-зависимости не нужны.

## Безопасный первый запуск

Сначала проверьте список углов без подключения к ОПУ и без движения:

```bash
python3 techlaser_azimuth_scan.py 192.168.1.115 --start 0 --stop 90 --step 10 --dry-run
```

Замените `192.168.1.115` на IP-адрес вашего устройства.

После этого можно запустить медленное движение на небольшом секторе:

```bash
python3 techlaser_azimuth_scan.py 192.168.1.115 --start 0 --stop 90 --step 10 --speed 5 --dwell 1
```

При нажатии `Ctrl+C` скрипт отправляет команду остановки.

Если скрипт пишет `Axis is not ready, state=0`, ось не готова принимать команды позиционирования. Когда сектор движения безопасен, запустите самодиагностику:

```bash
python3 techlaser_azimuth_scan.py 192.168.1.115 --init-only
```

Если после этого появилось `Axis is ready`, запускайте движение с флагом `--init`:

```bash
python3 techlaser_azimuth_scan.py 192.168.1.115 --init --start 0 --stop 90 --step 10 --speed 5
```

## Примеры

Пройти азимут от 0 до 180 градусов с шагом 15 градусов:

```bash
python3 techlaser_azimuth_scan.py 192.168.1.115 --start 0 --stop 180 --step 15 --speed 5
```

Пройти полный круг почти до 360 градусов:

```bash
python3 techlaser_azimuth_scan.py 192.168.1.115 --start 0 --stop 359 --step 10 --speed 5
```

Двигаться в обратном направлении:

```bash
python3 techlaser_azimuth_scan.py 192.168.1.115 --start 90 --stop 0 --step -10 --speed 5
```

Если сервисный порт отличается от `9760`, укажите его явно:

```bash
python3 techlaser_azimuth_scan.py 192.168.1.115 --port 9760 --start 0 --stop 90 --step 10
```

## Параметры

- `host` - IP-адрес ОПУ.
- `--port` - TCP-порт сервисного протокола, по умолчанию `9760`.
- `--start` - начальный азимут в градусах.
- `--stop` - конечный азимут в градусах.
- `--step` - шаг между позициями в градусах.
- `--speed` - максимальная скорость перехода в градусах в секунду.
- `--dwell` - пауза после достижения позиции в секундах.
- `--tolerance` - допустимая ошибка позиционирования в градусах.
- `--poll` - период опроса текущей позиции в секундах.
- `--max-wait` - максимальное ожидание достижения одной позиции в секундах.
- `--init` - запустить самодиагностику оси перед движением и дождаться готовности.
- `--init-only` - только запустить самодиагностику и выйти без движения.
- `--dry-run` - вывести список углов без подключения к ОПУ.

## Используемые команды ОПУ

Скрипт использует сервисный ASCII-протокол ТехЛазер:

- `$a#` - получить состояние оси поворота.
- `$a,1#` - запустить самодиагностику оси поворота.
- `$b#` - получить флаги ошибок оси поворота.
- `$c#` - получить текущую позицию по азимуту.
- `$e#` - получить статус занятости оси.
- `$g#` - остановить ось поворота.
- `$j,targetPos,maxSpeed#` - перейти в позицию по азимуту.

## Важно

Перед первым реальным запуском убедитесь, что рядом с ОПУ нет людей и препятствий, полезная нагрузка надежно закреплена, а выбранный сектор движения безопасен.
